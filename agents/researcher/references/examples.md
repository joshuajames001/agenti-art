# Research Examples — Researcher

## Example 1: Quick Brief
**Request:** "What is LiteLLM?"

**Output:**
- LiteLLM is an open-source Python library that provides a unified API for calling 100+ LLM providers using the OpenAI format
- Supports Anthropic, OpenAI, Google, Cohere, Ollama, and more via a single `litellm.completion()` call
- Key features: load balancing, fallbacks, cost tracking, and a proxy server mode
- Actively maintained, 10k+ GitHub stars as of early 2025
- Best used when building provider-agnostic LLM applications

**Sources:**
- LiteLLM Docs (docs.litellm.ai) — official, high credibility
- GitHub anthropics/litellm — primary source, commit history verified

## Example 2: Standard Report
**Request:** "Research the Czech freelancer (OSVČ) market size and pain points"

**Executive Summary:**
The Czech OSVČ market comprises approximately 1.1 million registered self-employed individuals, representing ~20% of the workforce. Key pain points center on administrative burden, tax complexity, and cash flow management — areas significantly underserved by existing digital tools.

**Key Findings:**

*Market size:*
- ~1.1M active OSVČ registrations (Czech Statistical Office, 2024)
- Fastest growing segments: IT, creative services, consulting
- Average OSVČ spends 4-6 hours/month on invoicing and administration

*Pain points:*
- 67% cite tax and social insurance calculations as primary stress (survey, Podnikatel.cz)
- Invoice payment delays averaging 32 days affect cash flow for 80% of freelancers
- Limited banking and fintech products designed specifically for OSVČ needs

*Digital tool adoption:*
- Majority still use Excel or paper for invoicing
- Pohoda and Money S3 dominate but are designed for larger companies
- No dominant mobile-first solution exists for solo freelancers

**Contested Points:**
- Market growth estimates vary: some sources cite 3% YoY growth, others 7% — methodology differences

**Gaps & Unknowns:**
- No public data on OSVČ software spending per year
- Limited research on OSVČ in rural vs urban areas

**Sources:**
- Czech Statistical Office (czso.cz) — official government data, high credibility
- Podnikatel.cz survey 2024 — industry publication, medium credibility
- Ministry of Finance CZ (mfcr.cz) — official tax data, high credibility
